# Reviews — the coach's evaluation of my progress

The coach writes `review-<NNN>.md` here each run, based on what I logged in
`../my-progress/`. Each review is concise:

- **Did** — what I actually completed
- **Solid** — what I've clearly got
- **Gaps** — what's weak, missing, or skipped
- **One fix** — the single most important thing to correct next

This is where "am I missing anything?" gets answered. Empty until the first run after I
log progress.
