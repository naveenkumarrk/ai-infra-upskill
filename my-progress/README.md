# My Progress — what I actually did

**This folder is mine.** After each day's lesson, I add a file `day-<NNN>.md` here noting:

- ✅ What I finished (which coding task / hands-on / reading)
- 🤔 What confused me or I couldn't complete
- 🔗 Links to any code I wrote (commit, gist, repo)
- ⏱️ Roughly how long I spent

The coach reads new files here at the start of the next run, evaluates my progress,
flags anything I'm missing, and writes its assessment into `../reviews/`. The more
honestly I log here, the better it adapts.

Template:

```
# Day <NNN> — <date>
## Done
-
## Struggled with
-
## Code
-
## Time spent
```
