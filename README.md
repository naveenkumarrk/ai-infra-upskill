# AI Infra Upskill

A day-by-day system that trains me from early-career engineer to the **Staff+ AI
Infrastructure / Distributed Systems** bar for $300k+ global remote roles — **basics to
mastery**, 2–3 topics a day, and it **adapts to what I actually do**.

A cloud Claude Code routine runs every morning: it reads what I logged, evaluates my
progress, teaches the next topics, and pushes everything back here.

## How it works
- **[`ROADMAP.md`](./ROADMAP.md)** — the ordered basics→mastery syllabus (82 topics, 5 levels)
- **[`PROGRESS.md`](./PROGRESS.md)** — live tracker: what's done, where I am, what's next
- **[`COACH.md`](./COACH.md)** — how the coach teaches (lean format + adaptive loop)
- **[`PROFILE.md`](./PROFILE.md)** — who this is for
- **[`days/`](./days)** — the coach's daily lessons
- **[`my-progress/`](./my-progress)** — **I** log what I actually did each day
- **[`reviews/`](./reviews)** — the coach's evaluation of my progress + gap analysis

## The daily loop
1. Coach reads my latest `my-progress/` entry → writes an evaluation to `reviews/`
2. Teaches the next 2–3 topics (lean lesson) → `days/`
3. Updates `PROGRESS.md`
4. I study, then log what I did in `my-progress/` → feeds tomorrow's review
