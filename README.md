# AI Infra Upskill

A day-by-day, incremental curriculum to go from early-career engineer to the Staff+ AI
Infrastructure / Distributed Systems bar for $300k+ global remote roles.

A cloud Claude Code routine runs every morning, reads the coaching spec here, works out
which day of the plan it is, and writes that day's lesson into [`days/`](./days).

- **[`COACH.md`](./COACH.md)** — how the coach teaches (date logic, daily format, rules)
- **[`ROADMAP.md`](./ROADMAP.md)** — the 16-week curriculum
- **[`PROFILE.md`](./PROFILE.md)** — who this is for
- **[`days/`](./days)** — the accumulating daily lessons (one file per day)

Building in public. 🚀
